h1 -самый важный заголовок(например название книги)

h2-менее важный заголовок (например содержание книги)

h3- важность еще снижается (в содержании наименование произведения)

h4- важность еще ниже(список глав в произведении)

Тег DEL представляет диапазон текста, который был удалён из документа

Тег ins предназначен для выделения текста, который был добавлен в новую версию документа

Тег small уменьшает размер шрифта на единицу по сравнению с обычным текстом, допускается применения вложенных тегов small, при этом размер шрифта будет меньше с каждым вложенным уровнем

Тег b используется для выделения части текста, который надо выделить стиллистически, но не несет какого то важного значения,как правило выделен жирным шрифтом

Тег strong Кроме физического выделения жирным, добавляет семантическое. Выделенный текст будет считаться особенно важным для прочтения и понимания

Тег sup отображает текст в виде верхнего индекса 3 2

Тег sub отображает текст в виде нижнего индекса 3 2

Тег code предназначен для отображения одной или нескольких строк текста, который представляет собой программный код. Сюда относятся имена переменных, ключевые слова, тексты функции и т.д Браузеры обычно отображают содержимое контейнера как моноширинный текст уменьшенного размера.

Тег pre определяет блок в который помещают предварительно отформатированный текст. 

Браузер отображает     содержимое тега именно так, как вы его      напечатали, включая все          пробельные символы.

Тег span — интервал предназначен для выделения отдельных строк, символов или других строчных элементов для дальнейшего изменения их оформления с помощью стилей. Например, внутри абзаца можно изменить цвет и размер первого слова, если его выделить с помощью элемента span и задать для него желаемый стиль.
